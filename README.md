# Townia

Towniaは、Minecraftサーバーに国や町を作成し、土地を保護・管理できる強力なプラグインです。

Townyのようなチャンクベースの土地保護システムを採用しており、直感的な操作で町を発展させることができます。

※ このプラグインは開発中のため，バグや不具合が存在する可能性があります。使用する際は、必ずバックアップを取ることをお勧めします。

## 🌟 主な機能

- **町と国の設立**: プレイヤーは町を作り、市長として住民を招待し、領土を拡大できます。複数の町が集まって国を作ることも可能です。
- **土地の保護**: クレームされた土地では、町に所属していないプレイヤーによる破壊や建築がブロックされます。
- **経済連携**: 町や国の設立、維持、土地の購入にはサーバー内の通貨を使用します。税金を設定して住民から自動徴収することも可能です。
- **アクションバーとマップ**: 画面上のアクションバーで「今どこにいるか」がリアルタイムで表示されます。`/town map`を使えば、周囲の土地の所有状況をテキストマップで確認できます。
- **権限管理システム**: 町長、副町長、一般住民などの役職システムにより、細かい権限の設定が可能です。
- **フレンド機能**: 個人のフレンドリストを作成し、将来的にフレンドのみに特定のアクションを許可するなどの連携が可能です。

## 📜 コマンド一覧

### 🏘️ 町コマンド (`/town` または `/t`)
| コマンド                                 | 説明                            |
|--------------------------------------|-------------------------------|
| `/town [info] [町名]`                  | 自分、または指定した町の情報を表示します。         |
| `/town create <町名>`                  | 新しい町を設立します(費用がかかります)。         |
| `/town claim`                        | 現在立っているチャンクを町の領土として取得します。     |
| `/town unclaim`                      | 現在立っているチャンクを放棄(荒地に)します。       |
| `/town invite <プレイヤー>`               | 他のプレイヤーを町に招待します。              |
| `/town join <町名>`                    | 招待された町に参加します。                 |
| `/town leave`                        | 現在所属している町を脱退します。              |
| `/town kick <プレイヤー>`                 | 住民を町から追放します。                  |
| `/town deposit <金額>`                 | 町の銀行にお金を入金します。                |
| `/town withdraw <金額>`                | 町の銀行からお金を引き出します(市長権限が必要)。     |
| `/town rank add/remove <プレイヤー> <役職>` | 住民に役職(ASSISTANT等)を付与・剥奪します。   |
| `/town set spawn`                    | 現在地を町のスポーン地点に設定します。           |
| `/town set homeblock`                | 現在のチャンクを町のホームブロック(中心地)に設定します。 |
| `/town set name <新町名>`               | 町の名前を変更します。                   |
| `/town set board <メッセージ>`            | 町の掲示板(ログイン時などに表示)を設定します。      |
| `/town set taxes <金額>`               | 町の税金(住民から徴収する額)を設定します。        |
| `/town map`                          | 周囲のチャンクの所有状況をマップ形式で表示します。     |
| `/town here`                         | 現在立っているチャンクの土地情報を表示します。       |
| `/town list`                         | サーバー内に存在する町の一覧を表示します。         |
| `/town delete confirm`               | 町を解散・削除します。                   |

### 🗾 国コマンド (`/nation` または `/n`)
| コマンド                    | 説明                     |
|-------------------------|------------------------|
| `/nation [info] [国名]`   | 自分、または指定した国の情報を表示します。  |
| `/nation create <国名>`   | 自分の町を首都として、新しい国を設立します。 |
| `/nation invite <町名>`   | 他の町を国に招待します。           |
| `/nation join <国名>`     | 招待された国に参加します。          |
| `/nation leave`         | 現在所属している国を脱退します。       |
| `/nation deposit <金額>`  | 国の銀行にお金を入金します。         |
| `/nation withdraw <金額>` | 国の銀行からお金を引き出します。       |
| `/nation rename <新国名>`  | 国の名前を変更します。            |

### 🧑 住民コマンド (`/resident` または `/res`)
| コマンド                              | 説明                    |
|-----------------------------------|-----------------------|
| `/resident [info] [プレイヤー]`        | プレイヤーのステータス情報を表示します。  |
| `/resident friend add <プレイヤー>`    | プレイヤーをフレンドリストに追加します。  |
| `/resident friend remove <プレイヤー>` | プレイヤーをフレンドリストから削除します。 |
| `/resident friend list`           | フレンドリストを表示します。        |
| `/resident friend clear`          | フレンドリストを全消去します。       |

### 🏞️ 土地・プロットコマンド (`/plot`)
| コマンド                 | 説明                    |
|----------------------|-----------------------|
| `/plot info`         | 現在立っている土地の情報を表示します。   |
| `/plot claim`        | 販売されている土地を購入・取得します。   |
| `/plot unclaim`      | 個人の土地を放棄し、町の共有地に戻します。 |
| `/plot forsale <価格>` | 自分の土地を販売状態にします。       |
| `/plot notforsale`   | 土地の販売を取り消します。         |

### 👑 管理者コマンド (`/townia admin`)
※ OPまたは`townia.admin`権限が必要です。
- `/towniaadmin reload`: 設定ファイルや言語ファイルを再読み込みします。
- `/towniaadmin town delete <町名>`: 町を強制的に解散させます。
- `/towniaadmin nation delete <国名>`: 国を強制的に解散させます。
- `/towniaadmin unclaim`: 現在の土地を強制的に放棄させます。

...その他多数のコマンド

## ⚙️ セットアップと導入

1. **前提プラグイン**: [Vault](https://www.spigotmc.org/resources/vault.34315/)および対応する経済プラグイン(EssentialsX等)が必要です。
2. **インストール**: `Townia-x.x.x.jar`を`plugins`フォルダに入れ、サーバーを再起動します。
3. **データベース設定**:
   - デフォルトでは`plugins/Townia/database.db`(SQLite)が使用されます。
   - MySQLを使用する場合は`config.yml`内の`database.type`を`mysql`に変更し、接続情報を設定してください。
4. **言語設定**:
   - `config.yml`で言語ファイルを指定できます。日本語の場合は `messages/lang_ja.yml` が使用されます。

## 🔧 開発者向け

このプラグインはJava 21とPaper API(1.21)を使用して構築されています。
ビルドするには以下のコマンドを実行してください。

```bash
./gradlew build
```
ビルドされたJARファイルは `build/libs/` フォルダに出力されます。
